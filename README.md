# Palabras Encadenadas

## Requisitos previos

- [Node.js](https://nodejs.org/) (version 18 o superior recomendada)
- npm (se instala con Node.js)

## Instalacion

1. Clona el repositorio:
   ```bash
   git clone https://github.com/francois227/unq-ui-franco-paredes-trabajo-final.git
   ```

2. Ingresa a la carpeta del proyecto:
   ```bash
   cd unq-ui-franco-paredes-trabajo-final
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

## Ejecucion local

```bash
npm run dev
```

La aplicacion se ejecuta en `http://localhost:5173`
